## dijkstra-suite

A Dijkstra's algorithm implementation that aims to be simple to use and fast to run

## Quick Start

```sh
make run
```

if you want to autoreload upon changes:
*(NOTE: requires [watchexec](https://github.com/watchexec/watchexec) installed)*

```sh
make watch
```

## Run the app

```sh
cargo run
```
